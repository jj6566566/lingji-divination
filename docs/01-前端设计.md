## 设计风格（方向待定，可调整）

### 参考来源
- trae.cn — 暗色底、极客绿、硬朗直角、鼠标跟随光晕
- vercel.com — 简洁排版、留白、卡片层次
- linear.app — 暗色 UI 天花板、微动效

### 当前方向：现代暗色 + 东方神秘感

```
配色:
  底色:    深灰褐/玄青 (#1a1a1f ~ #0f0f14)
  卡片:    毛玻璃 (backdrop-blur, rgba(255,255,255,0.04))
  强调色:  朱砂红 (#c44b3c) 或 玄金 (#c9a96e)
  文字:    #e8e4dd (暖白)

字体:
  标题:   Noto Serif SC (宋体感，有气质)
  正文:   Inter / Noto Sans SC

氛围:
  - 暗色底 + 大量留白
  - 卦象居中，四周空旷，仪式感
  - 毛玻璃卡片悬浮
  - 背景可能有墨韵粒子/星空粒子
  - 鼠标跟随光晕 (Cursor Glow)

动效:
  - 铜钱旋转/滚动动画
  - 卦象揭示时的展开动效
  - AI 解读流式打字机
  - 页面切换平滑过渡
  - 滚动揭示 (scroll reveal)

气质:
  "不是算命摊，而是数字时代的卜者"
  沉稳、神秘、现代、克制
  不用大红大黄、不用书法字体满天飞
```

### 3. 六爻占卜流程

**步骤 1: 输入问题**
- 输入框 + 发送按钮
- 提示文案："集中注意力，默想你的问题"
- 动画：三枚铜钱静止状态

**步骤 2: 起卦（2秒动画）**
- 6 次投掷动画，每次显示：
  - 第几爻
  - 三枚铜钱正反面组合
  - 阴阳/老老类型
  - 是否变爻

**步骤 3: 展示卦象**
- 本卦 → 变卦（如果无变爻则无变卦）
- 卦名 + 卦象符号（䷊）+ 拼音
- 变爻位置标注
- 基础卦辞一行展示

**步骤 4: AI 流式解读**
- 打字机效果逐字出现
- Markdown 渲染
- 解读完成后显示操作按钮：
  - [保存] [复制] [再占一次] [追问]

### 4. 八字排盘流程

**步骤 1: 输入生日**
- 日期选择器（年月日时）
- 性别选择
- 地点（省份城市，用于校准真太阳时，可选）

**步骤 2: 排盘展示**
- 四柱：年柱 月柱 日柱 时柱
- 每柱显示天干 + 地支
- 十神标注
- 五行统计

**步骤 3: AI 流式解读**
- 同上流式效果

### 5. 塔罗占卜流程

**步骤 1: 选择牌阵**
- 单张 / 三张 / 凯尔特十字 等
- 每种牌阵的图示说明

**步骤 2: 洗牌 + 抽牌动画**
- 卡牌随机移动（洗牌）
- 用户点击选择指定数量的牌
- 翻牌动画

**步骤 3: 牌面展示 + AI 解读**

## 前后端通信

- 基地址: `http://localhost:8080/api/v1`
- 鉴权: Header `Authorization: Bearer <JWT>`
- 占卜: SSE 流式连接 `POST /api/v1/divination` → `text/event-stream`
- 其余: 普通 REST JSON
- Token 过期 → 全局拦截 401 → 跳登录页
