### 认证模块

```json
Request:  { "username": "xiaoming", "password": "123456", "email": "xx@xx.com" }
Response: { "code": 200, "message": "注册成功", "data": { "userId": 1 } }

```

```json
Request:  { "username": "xiaoming", "password": "123456" }

```

```json
Header:   Authorization: Bearer <旧Token>
Response: { "code": 200, "data": { "token": "新Token" } }
```

### 占卜模块（需 JWT 鉴权）

#### POST /divination
```
Content-Type: application/json
Header: Authorization: Bearer <JWT>

Request:  { "method": "liuyao", "question": "我该不该换工作？" }
校验:     method 必须在 [liuyao, bazi, tarot, meihua] 中
          question 1~500 字符

Response: text/event-stream (SSE)
event: hexagram
data: {"type":"hexagram","original":"地天泰","transformed":"雷天大壮",...}

event: interpretation
data: {"type":"text","content":"地天泰是吉卦..."}

event: done
data: {"type":"done","recordId":42,"tokens":{"prompt":800,"completion":300}}

event: error
data: {"type":"error","message":"AI服务超时，请重试"}
```


## 统一返回格式

```json
// 成功
{ "code": 200, "message": "成功", "data": { ... } }

// 业务异常
{ "code": 429, "message": "今日免费次数已用完（3/3）", "data": { "used": 3, "maxFree": 3 } }

// 参数校验失败
{ "code": 400, "message": "问题内容不能为空", "data": null }

// 服务器错误
{ "code": 500, "message": "服务器内部错误", "data": null }
```

## HTTP 状态码映射

| code | 含义 | 何时使用 |
|------|------|----------|
| 200 | 成功 | 正常返回 |
| 400 | 参数错误 | @Valid 校验失败、请求格式错误 |
| 401 | 未登录 | Token 过期/缺失 |
| 403 | 无权访问 | 查询别人的记录 |
| 404 | 不存在 | 记录ID不存在 |
| 429 | 超限 | 今日免费次数用完 |
| 500 | 服务器错误 | 未知异常（记录日志） |
| 503 | AI 不可用 | AI 服务超时或宕机 |

## 全局异常拦截

```
@RestControllerAdvice
┌─────────────────────────────────────┐
```

## 核心业务逻辑：一次占卜的 Service 流程

```java
public SseEmitter divine(Long userId, DivinationRequest req) {

    // 1. 检查配额（事务）
    DailyQuota quota = quotaService.checkAndIncrement(userId, LocalDate.now());
    if (quota == null) {
        throw new QuotaExceededException("今日次数已用完");
    }

    // 2. 创建占卜记录
    DivinationRecord record = new DivinationRecord();
    record.setUserId(userId);
    record.setMethod(req.getMethod());
    record.setQuestion(req.getQuestion());
    divinationMapper.insert(record);

    // 3. 调用 AI 服务
    SseEmitter emitter = new SseEmitter(120_000L); // 120秒超时

    taskExecutor.execute(() -> {
        try {
            // 3a. 请求 AI 服务
            AiDivineRequest aiReq = AiDivineRequest.builder()
                .method(req.getMethod())
                .question(req.getQuestion())
                .sessionId(userId + "_" + record.getId())
                .build();

            // 3b. 流式接收 AI 响应，同时转发给前端
            StringBuilder fullInterpretation = new StringBuilder();
            aiServiceClient.divineStream(aiReq, new StreamCallback() {
                @Override
                public void onHexagram(String hexagramData) {
                    emitter.send(SseEmitter.event()
                        .name("hexagram").data(hexagramData));
                }

                @Override
                public void onText(String chunk) {
                    fullInterpretation.append(chunk);
                    emitter.send(SseEmitter.event()
                        .name("interpretation").data(chunk));
                }

                @Override
                public void onDone(TokenUsage usage) {
                    // 3c. 保存完整解读
                    record.setInterpretation(fullInterpretation.toString());
                    record.setPromptTokens(usage.prompt);
                    record.setCompletionTokens(usage.completion);
                    divinationMapper.updateById(record);

                    emitter.send(SseEmitter.event()
                        .name("done").data(Map.of("recordId", record.getId())));
                    emitter.complete();
                }

                @Override
                public void onError(String error) {
                    emitter.send(SseEmitter.event()
                        .name("error").data(error));
                    emitter.completeWithError(new RuntimeException(error));
                }
            });

        } catch (Exception e) {
            // AI 调用失败 → 记录标记、配额回滚
            quotaService.rollback(userId, LocalDate.now());
            log.error("AI 服务调用失败, recordId={}", record.getId(), e);
            emitter.completeWithError(e);
        }
    });

    return emitter;
}
```

## 安全设计

- 密码: BCrypt 加密存储，明文永远不会出现在数据库
- JWT: 24小时过期，secret 从环境变量读取
- 垂直越权: 查询记录时校验 `record.userId == currentUserId`
- SQL 注入: MyBatis-Plus 默认预编译，使用 `#{}` 不用 `${}`
- CORS: 只允许前端域名跨域
